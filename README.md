## 자기소개
> ### **남다른 러닝 커브**를 통해 빠른 적응력을 보여주는 개발자 윤현입니다. 

**TABA 아카데미**에서 실무형 프로젝트를 수행하며, RESTful API 기반 백엔드 서버를 설계하고 Docker와 CI/CD를 이용한 배포 자동화를 경험했습니다.  

현재는 **오픈소스 AI 응용** 수업에서 CNN 기반 이미지 Segmentation 모델을 구축하는 프로젝트를 진행 중입니다.

ChatGPT를 필두로 발달한 LLM을 개발 및 업무 도구로 사용하는 것뿐만 아니라 효과적인 지식 습득을 위한 수단으로 사용하여 러닝 커브를 극적으로 단축하는 것이 장점입니다. 

복잡한 시스템 또는 코드 블록을 **Top-Down 방식**으로 해체하여 구조를 완전히 이해하는 사고방식을 바탕으로, 기술적 깊이와 효율적인 문제 해결을 동시에 추구하는 개발자로 성장하고자 합니다.
